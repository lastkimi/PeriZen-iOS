## 🌟 初见 · 极简的数字戒毒 (First Glance. Digital Detox.)
**余光 (PeriZen)** 是一款反直觉的极简坐姿与专注工具。它倡导“多巴胺脱敏”。
没有喧闹的弹窗，没有强制的打卡音效。当你戴上耳机或端坐于桌前，它只是你视线边缘的一抹微光。当你姿态偏离，轻微的物理震动会唤醒你的直觉。

> *“不要盯着我，用余光知道我的存在即可。”*

## ✨ 核心特性

- **余光感知 (Peripheral Awareness)**: 设定专属的专注时长（1-1440分钟），让时间在屏幕边缘安静流淌。
- **禅意静修 (Zen State Mode)**: 专属于个人的深度沉浸模式。严格监控坐姿，防止低头。
- **默契大厅 (Resonant Lobby)**: 跨越距离的云端连线。加入公共自习室或私密房间，感受他人存在的陪伴。
- **共享磁场 (Shared Magnetic Field)**: 体验默契自习的陪伴之力，同频共振，抵御孤独。
- **全面追踪 (Detailed Analytics)**: 精准的数据追踪，涵盖总专注时长、平均达标率与姿势偏离统计。

## 🚀 本地运行与开发
本项目基于 iOS / Swift 开发，结合硬件陀螺仪进行姿态追踪。

1. 克隆代码：
   ```bash
   git clone https://github.com/lastkimi/PeriZen-iOS.git
   cd PostureGuardiOS
   ```
2. 使用 Xcode 打开 `PostureGuardiOS.xcodeproj`。
3. 确保你的签名证书配置正确。
4. 运行应用并在实体 iPhone (需配备陀螺仪) 上进行真实姿态测试。
